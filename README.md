# bobbadilla
Tool for copying music files

## Usage
'''
bobbadilla
--verbose
--input-dir <top dir>
--m3u <m3u playlist file(s)>
--output-dir <top dir to put files>
--min-file-size (500KB)
--max-file-size <max individual file size (10MB)>
--max-data-volume <total data>
--max-length (10 min)
--min-length (1 min)
'''
